# Pooja Patel's Portfolio Website

A modern, responsive personal portfolio website built with HTML5, CSS3, and JavaScript, created as part of the Soft Nexis internship program.

## 🚀 Features

- **Responsive Design**: Works seamlessly on mobile, tablet, and desktop devices
- **Modern UI**: Clean and professional design with smooth animations
- **Three Pages**: Home, About, and Contact pages with consistent navigation
- **Mobile Menu**: Hamburger menu for mobile devices
- **Contact Form**: Functional contact form with validation
- **SEO Friendly**: Semantic HTML5 structure

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript (ES6+)
- Git & GitHub

## 📁 Project Structure
```
portfolio-site/
├── index.html          # Home page
├── about.html          # About page
├── contact.html        # Contact page
├── styles/
│   └── main.css        # Global stylesheet
├── scripts/
│   └── main.js         # JavaScript functionality
├── images/             # Image assets
└── README.md           # Project documentation
```

## 🎨 Design Features

- **Color Scheme**: Blue (#3498db), Dark Gray (#2c3e50), Red (#e74c3c)
- **Typography**: Segoe UI font family
- **Layout**: CSS Grid for projects, Flexbox for navigation
- **Responsive Breakpoints**: 768px (tablet), 480px (mobile)

## 🚀 Getting Started

### Prerequisites

- Web browser (Chrome, Firefox, Safari, etc.)
- Text editor (VS Code recommended)
- Git (for version control)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/poojapatel2406/portfolio-site.git
```

2. Navigate to project folder:
```bash
cd portfolio-site
```

3. Open `index.html` in your browser or use Live Server in VS Code

## 📱 Responsive Design

- **Desktop (>768px)**: Multi-column grid layout
- **Tablet (≤768px)**: Adjusted grid, hamburger menu
- **Mobile (≤480px)**: Single column, stacked content

## 🌐 Deployment

### GitHub Pages

1. Push code to GitHub
2. Go to Settings → Pages
3. Select branch: `main`
4. Your site will be live at: `https://poojapatel2406.github.io/portfolio-site`

### Netlify

1. Drag and drop project folder to [Netlify Drop](https://app.netlify.com/drop)
2. Get instant live URL
